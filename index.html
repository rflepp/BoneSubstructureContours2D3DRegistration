Automatic Multi-view X-ray/CT Registration
using Bone Substructure Contours

A novel, fully automated registration method achieving sub-millimeter accuracy for intraoperative bone registration.

Read Paper View Code
{/* Abstract Section */}
Abstract Accurate intraoperative X-ray/CT registration is essential for surgical navigation in orthopedic procedures. However, existing methods struggle with robustness or require manual steps. We propose a contour-based ICP optimization focusing on bone substructures to reduce ambiguity and operate fully automatically using only two X-ray images, achieving sub-millimeter accuracy and user-friendly integration into surgical workflows.
